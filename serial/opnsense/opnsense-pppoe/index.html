Cấu hình OPNsense quay PPPoE.

24 tháng 12, 2020 ( Cập nhật: 23 tháng 1, 2021) 
3456 • Comments
OPNsense Config
Xem qua loạt bài viết hướng dẫn cấu hình OPNsense tại đây
  • Đăng nhập vào OPNsense
    • IP mặc định là https://192.168.1.1 với Username là root và Password là opnsense
  • Vào mục Interface > WAN, Thiết lập theo bảng dưới. Những mục không được nêu thì để mặc định.
Mục Chọn Ghi chú
IPv4 Configuration type PPPoE
IPv6 Configuration type DHCPv6 Nếu không muốn bật IPv6 thì chọn None, tuỳ vào nhà mạng mà mục này sẽ khác nhau. (vd: Viettel sẽ là DHCPv6)
Username Username PPPoE do nhà mạng cấp
Password Password PPPoE do nhà mạng cấp
Ảnh chụp màn hình
Generic Configuration

PPPoE Configuraion
  • Bonus: Đối với nhà mạng Viettel, nếu bật IPv6 thì mục IPv6 Configuration Type sẽ chọn DHCPv6 và mục DHCPv6 client configuration sẽ thiết lập như bảng dưới.
Mục Chọn Ghi chú
Configuration mode Basic
Request only an IPv6 prefix Tick chọn
Prefix delegation size 64
Send IPv6 prefix hint Tick chọn
Use IPv4 connectivity Tick chọn
VLAN priority Disabled
Ảnh chụp màn hình
  • Nhấn SAVE. Vào mục Interfaces > Overview > chọn mục WAN Interface để xem chi tiết, nếu quay PPPoE thành công thì ở đây sẽ hiện các thông số như Gateway, IPV4/IPv6, …
